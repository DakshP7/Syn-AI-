# Syn-AI-
AI-powered health assistant with Hindi and English support, designed to provide simple and safe medical guidance for underserved communities in India.

🚀 Features
🤖 AI-based symptom analysis
🌐 Hindi + English support
⚠️ Emergency detection (112 / 108)
💬 Simple chat interface
🛠️ Tech Stack
Python
Streamlit
Groq API
💡 Purpose

This project aims to improve access to basic health information for people who may face language or accessibility barriers.

⚠️ Disclaimer

This tool is for informational purposes only and is not a substitute for professional medical advice.
